# Ultimate Emotion AI

AI based Face Emotion Detection Application.

This project detects human emotions using a live camera or a photo. It uses Artificial Intelligence to analyze facial expressions and identify emotions such as happy, sad, angry, surprise, fear, disgust, and neutral.

Developer: Bipul Kumar

---

## Features

• Real-time face detection using webcam  
• Emotion detection using AI  
• Photo emotion detection  
• Face bounding box detection  
• Emotion color indicators  
• Fast optimized detection  
• GUI based application  
• Splash screen with developer name
